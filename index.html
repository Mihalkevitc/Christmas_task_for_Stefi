Возведение в степень

Возведение в степень

— это операция, когда число (основание) умножается само на себя несколько раз. Это записывается как an, где:

  • a — основание степени;
  • n — показатель степени.

Основные правила:

  • am × an = am+n

  • (am)n = am×n

  • am ÷ an = am−n

  • a0 = 1 (если a ≠ 0)

  • a−n = 1/an

Пример: 23 = 2 × 2 × 2 = 8

Особенности:

  • 1 в степени: 1n = 1 для любого n
  • 0 в степени: 0n = 0 для любого положительного n
  • Отрицательные показатели: a−n = 1/an, если a ≠ 0

Многочлены

Многочлен

— это алгебраическое выражение, состоящее из переменных, коэффициентов и операций сложения, вычитания, умножения и возведения в степень.

Пример: 3x2 + 2x - 5

Основные действия с многочленами:

  • Сложение: Складываем подобные слагаемые.
    Пример: (3x2 + 2x) + (4x2 - x) = 3x2 + 2x + 4x2 - x = 7x2 + x.
  • Вычитание: Вычитаем одноимённые слагаемые.
    Пример: (5x2 - 3x + y) - (2x2 + x + y2) = 5x2 - 3x + y - 2x2 - x - y2= 3x2 - 4x - y2 + y
  • Умножение: Используем распределительный закон для умножения. Пример: (x + 3)(x - 2) = x2 - 2x + 3x - 6 = x2 + x - 6.
